  Portfolio

Tiny Search Engine

Written in C, the Tiny Search Engine replicates a web search engine and is comprised of three components: Web Crawler, Page Indexer, and Querier. Web Crawler is a standalone program that crawls the web and retrieves webpages starting from a “seed” URL. Page Indexer is a program that builds an index from a directory created by Web Crawler. Querier is standalone program that reads the index file produced by the Page Indexer, and page files produced by the Querier, and answers search queries submitted via stdin.
